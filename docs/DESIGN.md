# DESIGN.md

## Purpose
Capture the top-level design direction for Minakeep and point to the deeper design docs that define the real constraints.

## Current Design Themes
- cool monochrome studio rather than warm paper/archive tone
- notes-first public showroom instead of a hero-led landing page
- smaller, more human-scaled typography with neutral sans emphasis
- tighter professional desktop density in the owner workspace
- dynamic grid rhythm on the homepage, with varied card heights rather than rigid equal cards
- reusable design components and base CSS tokens so style changes do not require logic rewrites
- mobile-first responsive behavior that preserves fast reading and scanning
