# Public home showroom

## Goal
Turn the public homepage into a note-first showroom with dynamic grid rhythm and cleaner visual framing.

## Trigger / Entry
An anonymous visitor opens `/`.

## User-Visible Behavior
- The homepage is led by published note previews, not by a dominant marketing hero.
- Note previews appear in a grid arrangement with varied heights rather than rigid equal rows.
- The layout stays easy to scan quickly.
- Metadata remains compact and secondary to note titles and previews.

## Validation
- Published note previews dominate the first-screen experience.
- The homepage grid feels dynamic without becoming visually chaotic.
- Mobile layouts still scan cleanly.
- `npm run verify` passes.
